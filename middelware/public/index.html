Documentación

Api rest base url:
http://localhost:3000/

Realizando una búsqueda

  • Url:
    /api/items
  • Methods:
    [GET]
  • Parámetros:
    Se envía el string a buscar en una variable GET (query string) ?q=televisores
  • Devolución:
    Este endpoint retornará un json con los productos encontrados en base a la query realizada y un array de categorías con el cual crearemos el breadcrumb.

Accediendo a un producto específico

  • Url:
    /api/items/:ID
  • Methods:
    [GET]
  • Parámetros:
    Se le envía la ID del producto como parámetro en la url, por ejemplo /api/items/MLA670757536
  • Devolución:
    Devolverá la información extendida del producto seleccionado.
